# コネクト銭湯

## サイト概要
### サイトテーマ
銭湯の場所を検索したり、銭湯好きと繋がったりできるSNSサイト

### テーマを選んだ理由
上京して最初にハマったのが銭湯巡りなのですが、
近年銭湯の数が減少していることを知り、いろんな人に銭湯の良さを知ってもらい、
銭湯文化を盛り上げたいと考え、このテーマにしました。

### ターゲットユーザ
銭湯に行ってみたい人、銭湯が好きな人

### 主な利用シーン
銭湯を検索するとき、自分が行った銭湯を投稿するとき
